Εργασία - Virtual Library

Οδηγίες για την εγκατάσταση και την εκτέλεση της εφαρμογής
Ακολουθήστε τα παρακάτω βήματα για να εγκαταστήσετε τα απαιτούμενα modules, να τρέξετε τον server και να δοκιμάσετε τη λειτουργία της εφαρμογής:

1) Κατεβάστε τα modules:
Ανοίξτε το terminal και πλοηγηθείτε στον φάκελο που περιέχει το project. Εκτελέστε την εντολή:

npm install

2) Μετά την εγκατάσταση των modules, τρέξτε τον server με την παρακάτω εντολή:

node server.js

3) Πληροφορίες για τη βάση δεδομένων:
Η βάση δεδομένων books.sqlite περιέχει ήδη κάποια dummy entries προκειμένου να δοκιμάσετε τη λειτουργία του search book. Μπορείτε να προσθέσετε και άλλα βιβλία μέσω της φόρμας προσθήκης βιβλίου.

Σχετικά με την προσθήκη βιβλίου μέσω του webpage:
Όταν προσθέτετε ένα βιβλίο μέσω του webpage, γίνονται οι παρακάτω έλεγχοι:

-Ελέγχεται η συμπλήρωση όλων των πεδίων.
-Ελέγχεται η ορθή τιμή της τιμής του βιβλίου (price).
-Ελέγχεται αν το βιβλίο υπάρχει ήδη στη βάση δεδομένων.
